# Guess‑the‑Number Web App

A simple browser game built with HTML, CSS, and JavaScript.

- Select a range (10, 100, 1000)
- Guess the number with limited attempts
- Tracks guesses and shows win/lose modal

## Live Demo
[Play on GitHub Pages](https://godfann-gyening.github.io/guess-the-number-app/)  <!-- will work after Pages deploys -->

## Run locally
Open `index.html` in your browser.
